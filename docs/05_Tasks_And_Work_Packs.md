# Tasks and Work Packs

Tasks convert camp preparation into assignable work.

Work Packs convert tasks into useful outputs for people and teams.

## Task Statuses

- Draft
- Planned
- Assigned
- Accepted
- In Progress
- Ready for Check
- Complete
- Checked
- Blocked
- Cancelled
- Not Needed

## Work Packs

Work Packs can be generated by:

- person
- team
- phase
- mixed grouping
