# Database and Data Model

Camp is the root entity.

## Main Entities

- Camp
- Person
- Contact Link
- Team
- Team Membership
- Task
- Task Assignment
- Task Checklist Item
- Activity
- Programme Block
- Programme Session
- Form Template
- Form Field
- Form Response
- Form Response Field
- Document
- Communication Template
- Communication Log
- Readiness Requirement
- Export Record
- User Account
- Camp Access

## Key Decisions

- No permanent young person database.
- Contacts are person records.
- Programme Blocks organise schedules.
- Forms are structured data.
- Templates are copied, never live-linked.
