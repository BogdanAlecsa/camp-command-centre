# Imports and Exports

The app should make it easy to get data in and easy to get data out.

## Universal Import Wizard

All imports should use one common import engine.

Supported MVP import types:

- People
- Contacts
- Teams
- Tasks
- Activities
- Programme
- Forms
- Documents

## Export Types

- Camp overview
- People list
- emergency contacts
- team lists
- task sheets
- team work packs
- programme
- group schedules
- leader schedules
- parent pack
- leader pack
- readiness report
