# How The App Works

Camp Command Centre is organised around a Camp.

A Camp contains people, teams, programme blocks, activities, tasks, forms, documents, communications, readiness checks and outputs.

```text
Camp
 ├── People
 ├── Teams
 ├── Forms
 ├── Programme
 ├── Tasks
 ├── Documents
 ├── Communications
 ├── Readiness
 └── Outputs
```

## Core Flow

```text
Create Camp
Add People
Collect Information
Create Teams
Build Programme
Assign Tasks
Check Readiness
Generate Packs
```
